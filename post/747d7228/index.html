信息

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行时

  • 官网：https://nodejs.org/zh-cn/
  • 官方文档：https://nodejs.org/zh-cn/docs/
  • 中文文档：http://nodejs.cn/api/

安装

  1. 进入官网，下载安装包，安装

    安装过程中可以改安装路径
    它会自动给系统添加环境变量

  2. 测试
    cmd中输入npm得到一堆信息，则说明安装成功

  3. 配置node_cache路径
    这个文件夹是Node.js用来存放下载包的缓存的地方

  4. 设置全局模块存放路径
    这个文件夹是Node.js用来存放通过全局安装的模块的地方
    Node.js会自动寻找该路径下的node_modules文件夹为实际存放全局模块的路径
    我一般设置为在Node.js目录

    node_modules文件夹为实际存放全局模块的路径，这也是为啥叫prefix不叫global的原因

  5. 修改npmrc文件
    用文本编辑器打开Node.js目录\node_modules\npm\npmrc文件
    将prefix=${APPDATA}\npm改为prefix=Node.js根目录
    如果不做这个修改，则在运行npm ls -g的时候，仍然以默认的路径来查找已安装的全局模块
