VSCode-IDE

本文最后更新于：2021年4月22日 下午

Visual Studio Code

轻量级，相当于notepad++加强版，很多东西要自己配，各种平台均可用

地址

官方文档：https://code.visualstudio.com/docs/setup/linux
下载地址：https://visualstudio.microsoft.com/zh-hans/

安装

windows

直接使用安装包即可

CentOS

https://code.visualstudio.com/docs/setup/linux
已经将 稳定的64位的VSCode放在了yum里了
你可以用下面的代码安装秘钥和库

  • 安装方法一：用dnf,更新cache包,安装对应的包
  • 安装方法二：用yum,更新cache包,安装对应的包 需要注意的是，用yum安装的话，VSCode的版本可能会滞后。

常用快捷键

键位 作用
Ctrl / 注释代码
Ctrl [ 代码行向左缩进
Ctrl ] 代码行向左缩进
Shift Alt F 代码格式化
Shift F12 找到所有的引用
F11 全屏显示(再次按则恢复)
Ctrl S 保存
F1 打开VSCode命令面板

插件

安装插件


安装插件简单三步：点插件，输入插件名称，点击安装

常用插件

插件名 作用
Chinese (Simplified) Language Pack for Visual Studio Code 界面汉化
Beautify 美化javascript, JSON, CSS, Sass，和HTML代码
GitLens — Git supercharged 增强git在Visual Studio中的功能
vscode-icons 在Visual Studio文件浏览中加入图标，方便识别
Python python运行环境插件
Import Cost 可以计算import的消耗
RegExp Preview and Editor 正则表达式可视化处理工具
